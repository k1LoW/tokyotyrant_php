h2. 対応状況

現在tokyotyrant.rb Ver 1.6の

* TokyoTyrant::RDB
* TokyoTyrant::RDBTBL
* TokyoTyrant::RDBQRY

の約90%はカバーしていると考えています。
現在タスクとして残っているのはPHP固有の問題の解消のみです。

h2. PHP固有の問題

現在分かっているPHP固有の問題と対応状況は以下です。

* 64bitの整数の扱い
** 対応:floatで返している
* constにビットシフトの式を含む値を設定することができない。
** 対応:コメントアウト、または整数を設定
